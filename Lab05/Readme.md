Опис проекту
Lab2 — це комплексний десктопний додаток на базі WPF (C#), призначений для автоматизації широкого спектра операцій лінійної алгебри та розв'язання задач лінійного програмування (ЗЛП). Програма поєднує в собі інструменти для роботи з матрицями та аналітичний модуль для пошуку екстремумів функцій.

Основна мета роботи — опанувати використання модифікованих жорданових виключень (МЖВ) для розв’язання задач оптимізації із системою обмежень у вигляді нерівностей та рівностей. Основний функціонал

1. Лінійна алгебра (Частина А)

– Знаходження рангу матриці;

– Обчислення оберненої матриці;

– Розв’язання СЛАР.

2. Лінійне програмування (Частина B)

– Оптимізація цільової функції: Пошук максимуму та мінімуму функції при заданих обмеженнях (нерівностях);

– Пошук опорного розв’язку: Автоматична побудова початкового допустимого базису для ЗЛП;

– Симплекс-метод: Реалізація ітераційного процесу пошуку оптимального розв’язку на основі МЖВ.


3. Лінійне програмування. Змішана система обмежень в задачі.

– Оптимізація цільової функції: Пошук максимуму та мінімуму функції при заданих обмеженнях (рівностях та нерівностях);

– Пошук опорного розв’язку: Автоматична побудова початкового допустимого базису для ЗЛП;

– Симплекс-метод: Реалізація ітераційного процесу пошуку оптимального розв’язку на основі МЖВ.

4. Лінійне програмування. Цілочисельне розв'язання задачі.

– Оптимізація цільової функції: Пошук максимуму та мінімуму функції при заданих обмеженнях (рівностях та нерівностях);

– Пошук опорного розв’язку: Автоматична побудова початкового допустимого базису для ЗЛП;

– Симплекс-метод: Реалізація ітераційного процесу пошуку цілочисельного оптимального розв’язку на основі МЖВ.

5. Розв'язання двоїстої задачі.

– Оптимізація цільової функції: Пошук максимуму та мінімуму функції при заданих обмеженнях (рівностях та нерівностях);

– Пошук опорного розв’язку: Автоматична побудова початкового допустимого базису для ЗЛП;

– Симплекс-метод: Реалізація ітераційного процесу пошуку цілочисельного оптимального розв’язку на основі МЖВ.

– Пошук розв'язку двоїстої задачі за допомогою модифікованих жорданових виключень.

Технології

– Мова: C# (.NET 6 / .NET 8)

– UI Framework: WPF (Windows Presentation Foundation)

– Середовище розробки: Visual Studio 2022
