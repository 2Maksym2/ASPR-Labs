Опис проекту
Lab1 — десктопний додаток на WPF (C#) для виконання операцій лінійної алгебри. Програма дозволяє розв'язувати системи лінійних алгебраїчних рівнянь (СЛАР), знаходити обернену матрицю та обчислювати ранг матриці, використовуючи звичайний метод Жорданових виключень.


Ключова особливість — можливість збереження детального протоколу обчислень у файл.

Функціонал
- Знаходження рангу матриці.

- Розв'язання СЛАР.

- Обчислення оберненої матриці.

- Генерація протоколу: автоматичне збереження кроків розв'язку у файл (Protocol.txt).

- Обробка винятків.

Технології
- Мова: C# (.NET 6 / .NET 8)
- UI Framework: WPF (Windows Presentation Foundation)
- Середовище розробки: Visual Studio 2022
